# SMAJ Business Landing Page

Welcome to the SMAJ Business Landing Page project! This repository contains the code for a responsive, animated single-page website built using **HTML**, **CSS**, and **JavaScript**. 

## Features

1. **Modern Layout**
   - Sleek and responsive design, ensuring optimal viewing on all devices.
   
2. **Hero Section**
   - Engaging welcome message with smoothly animated transitions, including a profile picture.

3. **Services Section**
   - Highlight of business services offered, with clean and organized cards.

4. **Contact Form**
   - Easy-to-use contact form for your visitors to reach out to you.

5. **Social Media Integration**
   - Links to Facebook, TikTok, LinkedIn, Instagram, Telegram, Twitter, and Discord with icons powered by [Boxicons](https://boxicons.com/).

## Built With
- **HTML**: For the structure.
- **CSS**: For styling and appearance.
- **JavaScript**: For interactivity and animations.
- [Animate.css](https://animate.style/): For smooth animations.
- [Boxicons](https://boxicons.com/): For social media icons.
  
## How to Use

### Prerequisites
1. A web browser - Recommended: Chrome, Firefox, Edge.
2. A simple text editor or development environment - Recommended: VSCode, Sublime Text.

### Steps
1. Clone the repository:
    ```bash
    git clone https://github.com/smajceo/SMAJ-Busines-landing-page.git
    ```
2. Open the project folder, and start a server to serve the files locally (optional).
3. Open `index.html` in your browser to view the landing page.
